================================================= To run this program =================================================
1. Change dotenv_path to correct env file path in testingGrammarModel.py, uploadFile.py, and grammar_transformer.py.
2. cd template
3. run run.py file (python3 run.py/ python run.py )
4. Go to: http://127.0.0.1:5000
5. upload a.docx file (example file) to test Continue Your Story Generator, Story Outline Generator
========================================================================================================================

=================================== If you want to use Grammar Checking & Correction ===================================
1. Change http://127.0.0.1:5000 -----> http://127.0.0.1:5001
2. To test this function: 
                          a. Go to readyToGoGrammarTrainResources/GTP3TrainningResource/grammar_correction_output.txt
                          b. Copy and paste the Original paragraph from grammar_correction_output.txt
                          c. The correct output is in the Corrected paragraph

========================================================================================================================

=================================================== About the GPT KEY ===================================================
1. This project needs GPT KEY to be active, which is your env file.
2. Please use your own GPT KEY
This env file should be like this:
OPENAI_API_KEY = YOUR_OWN_OPENAI_API_KEY
